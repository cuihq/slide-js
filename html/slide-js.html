1. html

这是最简单的一种形式。

优点 缺点
简单、直接、粗暴 不符合表现与内容分开的原则

例子：

<img src="/favicon.ico">文字

2. CSS

这是最常用的一种形式。

优点 缺点
可控制图标的更多的显示样式 不够直观

例子：

<span class='icon'></span>文字
.icon {
  display: inline-block;
  background: url(/favicon.ico) no-repeat;
  width: 32px;
  height: 32px;
}

3. inline image

根据RFC 2397标准，inline image的格式如下:

data:[<mediatype>][;base64],<data>

优点 缺点
减少HTTP请求 可维护性，可读性，兼容性差

例1:

<img src="data:image/x-icon;base64,AAABAAEAICA..." />

例2：

.icon {
  width:32px;
  height:32px;
  background-image: url(data:image/x-icon;base64,AAABAAEAICA...);
}

资源和工具

websemantics, 免费在线图片转换

4. sprint image

将多个图标合成一个图片，并用CSS控制。

优点 缺点
减少HTTP请求 放大光标失真

例子：

.icons-adduser, .icons-delete {
  background: url('/icons.png') no-repeat;
}

.icons-adduser {
  background-position: 0 0;
}

.icons-delete {
  background-position: 0 -16px;
}

5. icon fonts

将图标编码成字体，并使用CSS进行控制。

优点 缺点
灵活，方便，减少HTTP请求 不支持彩色图片

例子：

@font-face {
  font-family: "font-icon";
  src: url("/font-icon.woff");
}
.search:before {
  content: "s";
  display: block;
  font-family: font-icon;
}
<div class="search">搜索</div>

资源和工具：

fontsquirrel，有一些免费字体图标

FontCreator，可制作自定义字体图标
